# Portfolio — Muhammad Arya Bintana

## Deploy ke Vercel

1. Upload semua file ini ke GitHub
2. Connect repo ke Vercel → Deploy otomatis

## Cara Tambah Screenshot

Letakkan gambar di folder:

```
public/
└── screenshots/
    ├── web-masjid/
    │   ├── 1.jpg   ← Beranda
    │   ├── 2.jpg   ← Fitur
    │   ├── 3.jpg   ← Detail
    │   ├── 4.jpg   ← Tampilan
    │   └── 5.jpg   ← Lainnya
    ├── ppdb-nurul-ilmi/
    │   ├── 1.jpg
    │   └── ... (sampai 5.jpg)
    ├── makeupmarcos/
    ├── kabarkini/
    └── rumah-kita/
```

**Format gambar:** .jpg atau .png (rename jadi .jpg)  
**Resolusi recommended:** 1280x800px ke atas  
**Jumlah per proyek:** Bisa 1–5 gambar. Yang tidak ada otomatis pakai placeholder.
